# 5. Створити клас ЛІКИ, які містять: назву, кількість одиниць товару, вартість. Визначити вартість ліків за рецептом. Перевантажити операції (+=, -=) для зміни кількості одиниць товару, (+, -) для зміни вартості товару. Первантажити методи вартості ліків за рецептом та пошуку ліків за різними критеріями. Передбачити обробку виняткових ситуацій при встановленні та читанні числових полів класу ЛІКИ.
